P47s - Product/Variant Image Update Not Persisting (root-cause fixes)

Scope
- src/app/dashboard/products/new/page.tsx
- src/app/dashboard/products/[id]/edit/page.tsx
- src/services/products.ts

Root causes (confirmed in code)
1) Variant images are not wired to the form at all.
   - Both new/edit pages render ImageUploader but only console.log the result.
   - So variant image changes are never sent to the server.

2) Clearing all product images does not persist.
   - updateProduct only sets dataToUpdate.image when finalImages.length > 0.
   - If you delete all images, finalImages = [], so DB is never updated.

Apply the fixes exactly below.

------------------------------------------------------------
1) Frontend: wire variant images to form + allow file uploads
------------------------------------------------------------

We will:
- Bind variant image to `variations.${index}.image`.
- Track variant image files in a ref, and send them via FormData as `variantImage:<variationId>`.
- Server will upload these files and replace the temporary marker.

A) New product page

File: src/app/dashboard/products/new/page.tsx

1. Add ref and helper inside component (near other hooks):

```ts
const variantImageFilesRef = React.useRef<Record<string, File>>({});

const setVariantImageFile = (variationId: string, file: File | null) => {
  if (!variationId) return;
  if (!file) {
    delete variantImageFilesRef.current[variationId];
    return;
  }
  variantImageFilesRef.current[variationId] = file;
};
```

2. In the variation image UI block, REPLACE this line:

```ts
<ImageUploader onImagesChange={(files) => console.log('variation image', files)} isMultiple={false} images={[]} />
```

WITH this block:

```tsx
<FormField
  control={form.control}
  name={`variations.${index}.image`}
  render={({ field }) => {
    const variationId = form.getValues(`variations.${index}.id`);
    const currentUrl = typeof field.value === 'string' && field.value && !field.value.startsWith('__file__:')
      ? field.value
      : '';

    const images = currentUrl ? [{ id: currentUrl, url: currentUrl }] : [];

    return (
      <FormItem>
        <FormControl>
          <ImageUploader
            isMultiple={false}
            images={images}
            onImagesChange={(files) => {
              const first = files[0];

              // cleared
              if (!first) {
                setVariantImageFile(variationId, null);
                field.onChange('');
                return;
              }

              // file upload -> store file, set temp marker
              if (first instanceof File) {
                setVariantImageFile(variationId, first);
                field.onChange(`__file__:${variationId}`);
                return;
              }

              // library url
              setVariantImageFile(variationId, null);
              field.onChange(first.url);
            }}
          />
        </FormControl>
        <FormMessage />
      </FormItem>
    );
  }}
/>
```

3. In onSubmit, after these lines:

```ts
formData.set('attributes', JSON.stringify(values.attributes || []));
formData.set('variations', JSON.stringify(values.variations || []));
formData.set('productType', values.productType || 'simple');
```

ADD:

```ts
Object.entries(variantImageFilesRef.current).forEach(([variationId, file]) => {
  formData.append(`variantImage:${variationId}`, file);
});
```

B) updateProduct: upload variantImage:* files and apply to variations

Add this block JUST BEFORE the "Variations (rebuild)" section:

```ts
// -------- Variant image uploads (per-variant) --------
const variantImageEntries = Array.from(formData.entries()).filter(
  ([key, value]) => key.startsWith('variantImage:') && value instanceof File,
) as [string, File][];

const uploadedVariantImageMap = new Map<string, string>();

if (variantImageEntries.length > 0) {
  for (const [key, file] of variantImageEntries) {
    try {
      const variationId = key.split(':')[1];
      const uploaded = await uploadImages([file]);
      const url = normalizeUrl(uploaded[0]?.url);
      if (variationId && url) {
        uploadedVariantImageMap.set(variationId, url);
      }
    } catch (err) {
      console.warn('[VARIANT_IMAGE_UPLOAD_FAILED]', key, err);
    }
  }
}
```

Then, inside the variations loop(s), compute image like this:


```ts
const uploadedVariantUrl = variant?.id ? uploadedVariantImageMap.get(String(variant.id)) : undefined;
const imageValue = uploadedVariantUrl
  ?? (typeof variant.image === 'string' && !variant.image.startsWith('__file__:') ? variant.image : null);

image: imageValue,
```

Do this in BOTH places:
