P47ad - Commission only on Delivered + Incomplete Convert rate + Delivery/Return ratios (per staff)

Goal
- Staff commissions (create/confirm/convert) are paid ONLY when the order reaches Delivered.
- Incomplete conversions can have a separate commission rate (fallback to create rate if not set).
- Show Delivery/Return ratios per staff:
  1) Creation pipeline ratio (orders created by this staff → delivered/returned)
  2) Confirmation pipeline ratio (orders confirmed by this staff → delivered/returned)
  3) Total ratio (combined created+confirmed delivered/returned) in compact form

Confirmed by lead
- Ratio formula:
  deliveryRatio = delivered / (delivered + returned)
  returnRatio = returned / (delivered + returned)
- Ratios must be calculated per staff (their own counts).
- Need both: creation pipeline ratios + confirmation pipeline ratios + total ratio (compact).
- No backfill for historical orders.
- Commission for orders that never reach Delivered should be zero.
- If an order later becomes Returned/Canceled/Damaged, delete any commission entries for that order.

2b) Add new helper: awardCommissionOnDelivered
Pseudo:
```
async function awardCommissionOnDelivered(tx, orderId) {
  const order = await tx.order.findUnique({
    where: { id: orderId },
    select: { createdBy: true, confirmedBy: true, source: true }
  });
  if (!order) return;

  const existing = await tx.staffIncome.findMany({
    where: { orderId },
    select: { staffId: true, action: true }
  });
  const existingKey = new Set(existing.map(x => `${x.staffId}:${x.action}`));

  // Created / Converted
  if (order.createdBy) {
    const staff = await tx.staffMember.findUnique({
      where: { id: order.createdBy },
      select: { commissionDetails: true }
    });
    const isConvert = order.source === 'woo-incomplete';
    const key = isConvert ? 'onOrderConvert' : 'onOrderCreate';
    const fallback = isConvert ? 'onOrderCreate' : null;
    const amount = getCommissionAmount(staff?.commissionDetails, key) 
      || (fallback ? getCommissionAmount(staff?.commissionDetails, fallback) : 0);
    if (amount > 0 && !existingKey.has(`${order.createdBy}:Created`)) {
      await recordStaffIncome(tx, { staffId: order.createdBy, orderId, action: 'Created', amount, notes: isConvert ? 'Converted from incomplete' : undefined });
    }
  }

  // Confirmed
  if (order.confirmedBy) {
    const staff = await tx.staffMember.findUnique({
      where: { id: order.confirmedBy },
      select: { commissionDetails: true }
    });
    const amount = getCommissionAmount(staff?.commissionDetails, 'onOrderConfirm');
    if (amount > 0 && !existingKey.has(`${order.confirmedBy}:Confirmed`)) {
      await recordStaffIncome(tx, { staffId: order.confirmedBy, orderId, action: 'Confirmed', amount });
    }
  }
}
```

UI computation (both Staff Detail + My Account):

Creation pipeline ratio:
```
const cDelivered = createdStatusBreakdown['Delivered'] || 0;
const cReturned = createdStatusBreakdown['Returned'] || 0;
const cTotal = cDelivered + cReturned;
const cDeliveryRatio = cTotal ? (cDelivered / cTotal) * 100 : 0;
const cReturnRatio = cTotal ? (cReturned / cTotal) * 100 : 0;
```

Confirmation pipeline ratio:
```
const fDelivered = confirmedStatusBreakdown['Delivered'] || 0;
const fReturned = confirmedStatusBreakdown['Returned'] || 0;
const fTotal = fDelivered + fReturned;
const fDeliveryRatio = fTotal ? (fDelivered / fTotal) * 100 : 0;
const fReturnRatio = fTotal ? (fReturned / fTotal) * 100 : 0;
```

Total ratio (compact):
```
const tDelivered = cDelivered + fDelivered;
const tReturned = cReturned + fReturned;
const tTotal = tDelivered + tReturned;
const tDeliveryRatio = tTotal ? (tDelivered / tTotal) * 100 : 0;
const tReturnRatio = tTotal ? (tReturned / tTotal) * 100 : 0;
```
