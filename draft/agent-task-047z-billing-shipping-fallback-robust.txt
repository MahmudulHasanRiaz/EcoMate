P47z - Woo Blocks: Billing/Shipping Fallback Robust Capture

Goal
- Default Woo (Blocks) checkout should capture full fields (name/address/etc), not only phone.
- Make shipping vs billing fallback explicit and bug-resistant.

Root cause
- Blocks often uses `shipping-*` field names when "Use same address for billing" is enabled.
- Our selectors and extraction are still too strict (underscore-only + explicit selectors only).

Scope
- Only update the plugin JS:
  - draft/fashionary-woo-plugin/assets/js/checkout.js
- Then bump plugin version + rebuild zip.

2) Add a generic billing/shipping fallback extractor (critical fix)
----------------------------------------------------------------------
We will keep explicit selectors, but add a robust fallback that:
- Prefers billing fields when filled
- Falls back to shipping fields
- Falls back to any matching field name/id

In checkout.js, add these helpers below `getVal()`:

```js
function readValue($el) {
    if (!$el || !$el.length) return '';
    if ($el.is(':disabled')) return '';
    const v = String($el.val() || '').trim();
    return v;
}

function collectCandidates(prefix, keywords) {
    const sels = [];
    for (const kw of keywords) {
        sels.push(
            `input[name*="${prefix}"][name*="${kw}"]`,
            `input[id*="${prefix}"][id*="${kw}"]`,
            `select[name*="${prefix}"][name*="${kw}"]`,
            `select[id*="${prefix}"][id*="${kw}"]`,
            `textarea[name*="${prefix}"][name*="${kw}"]`,
            `textarea[id*="${prefix}"][id*="${kw}"]`
        );
    }
    return $(sels.join(', '));
}

function fallbackByKeywords(keywords) {
    const billing = collectCandidates('billing', keywords);
    const shipping = collectCandidates('shipping', keywords);

    // 1) Try billing filled fields first
    for (const el of billing.toArray()) {
        const v = readValue($(el));
        if (v) return v;
    }

    // 2) Then shipping
    for (const el of shipping.toArray()) {
        const v = readValue($(el));
        if (v) return v;
    }

    // 3) Finally, any field containing the keywords
    const genericSels = [];
    for (const kw of keywords) {
        genericSels.push(
            `input[name*="${kw}"]`,
            `input[id*="${kw}"]`,
            `select[name*="${kw}"]`,
            `select[id*="${kw}"]`,
            `textarea[name*="${kw}"]`,
            `textarea[id*="${kw}"]`
        );
    }
    const generic = $(genericSels.join(', '));
    for (const el of generic.toArray()) {
        const $el = $(el);
        if ($el.attr('type') === 'hidden') continue;
        const v = readValue($el);
        if (v) return v;
    }

    return '';
}

function getValSmart(selectors, keywords) {
    // Explicit selectors first
    const explicit = getVal(selectors);
    if (explicit) return explicit;
    // Keyword-based fallback
    return fallbackByKeywords(keywords);
}
```
