2) Guard against null detail in openConvertModal
File: src/app/dashboard/orders/incomplete/page.tsx

Update openConvertModal:

From:
```ts
const detail = await getIncompleteLead(orderId);
setConvertTarget(detail);
setConvertForm({
  name: detail.name || '',
  phone: detail.phone || '',

});

```

To:
```ts

setConvertTarget(detail);
setConvertForm({
  name: detail.name || '',
  phone: detail.phone || '',
  address: detail.address || '',
  skuCsv: skuListToCsv(detail.skuList),
});
setIsConvertOpen(true);
```
