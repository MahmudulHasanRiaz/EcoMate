UPDATE 11 - Invoice layout polish + compact lot display

Goal: improve invoice readability with less space usage. No data logic changes.
Focus: handleGenerateWorkOrder in src/app/dashboard/purchases/[id]/client-page.tsx

Design rules
- Keep layout consistent across Fabric / Printing / Cutting / Finishing.
- Reduce vertical space; avoid extra rows for lot allocations.
- Use compact inline lot summaries per item.

1) Remove extra image column (fabric/printing/cutting)
- Currently rows include an image <td> but headers do NOT include an Image column.
- Move image into the Product cell as a small inline block (left of name).
- Then Product header aligns with data; no extra columns.

Implementation pattern (inline in Product cell):
- Wrap with <div style="display:flex;gap:8px;align-items:flex-start;">...
- Image size ~32px, border 1px, border-radius 4px.
- Product name + variant in a nested <div>.

2) Compact lot allocation display (internal fabric only)
- Remove the extra "Lot Allocation" table row (pindaRow) entirely.
- Instead, add a small, muted line under product name inside the Product cell:
  "Lots: L001 U:10 R:5 | L002 U:7 R:0"
- Use small font (10px), muted color (#64748b), and allow wrap.
- If there are no lots, do not show the line.

Optional compact overflow (if many lots):
- If allocations.length > 3, show first 3 and append "+N more".

3) Column widths + table density
- Use table styles to reduce space:
  - font-size: 12px; line-height: 1.3;
  - th/td padding: 6px 8px;
  - table-layout: fixed;
- Set column widths via <th style="width:..."> for consistency.

4) Damage/Wastage lines
- Keep damage lines small and compact.
- Use consistent label "Dmg: X" in qty cell (printing/cutting).
- For cutting, keep "For X pcs" if (printing+cutting damage) > 0 (already done).

5) Summary block alignment
- Keep summary block on the right with less width (e.g., 45%).
- Add `margin-left:auto;` so it aligns right and saves space.

6) Finishing invoice
- Keep columns: Product | Wastage | Final Qty | Unit Cost | Line Total.
- Make Wastage and Final Qty narrow (width 10% each).
