- This causes TS/runtime failure.

Required implementation pattern

   - `let effectivePrice = price;`  // default Woo unit price
2. Keep `siteDiscount` initialized to 0.
3. Combo branch:
   - Compute `comboUnitPrice` from component unit prices.
   - Compute `comboGross = comboUnitPrice * quantity`.

   - Resolve sale/regular effective unit as current logic intends.
   - `effectivePrice = resolvedEffectivePrice`

5. Return payload uses:
   - `price: effectivePrice`
   - `siteDiscount`
